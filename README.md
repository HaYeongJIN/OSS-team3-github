## [Git 기초 개념 및 명령어](git-concept-command.md)

Git은 프로젝트의 상태 변화를 '스냅샷' 형태로 기록하는 분산 버전 관리 시스템입니다. 단순히 파일을 저장하는 것을 넘어, 시간의 흐름에 따라 누가, 언제, 무엇을 바꿨는지 추적할 수 있게 해줍니다.

- 핵심 명령어 요약:
  - `git init`: 현재 폴더를 Git이 관리하는 저장소로 초기화합니다.
  - `git status`: 파일의 수정 여부와 '준비 상태'를 실시간으로 확인합니다.
  - `git add`: 변경된 파일 중 다음 저장 기록(Commit)에 포함될 것들을 골라냅니다.
  - `git commit`: 선택된 변경 사항을 하나의 기록으로 영구히 저장합니다.
  - `git log`: 지금까지 쌓인 프로젝트의 모든 변경 이력을 확인합니다.
